# Install 필요한 패키지

```
pip install markdown
```

# 프로그램 기능

- markdown (.md) 형식의 파일을 .html 파일로 변환
- 테이블, 코드 하이라이팅, 목차, 줄바꿈 등 지원
- CSS 스타일 포함

# 실행 방법

```
# 기본 사용 (출력 파일명 자동 생성)
python markdown_converter.py example.md

# 출력 파일명 직접 지정
python markdown_converter.py example.md output.html
```
